## Installation
1. Clone the repo
   ```sh
   git clone https://github.com/helloimmichelle/FYP-word-based-game.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run in expo
   ```sh
   npx expo start
   ```
